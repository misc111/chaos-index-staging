The model room, without the math fog.

Each model is a different kind of scout. Some are steady actuaries, some are curve readers, and some are experimental scouts that are only here to challenge the room.

Plain ideaWhen it helpsHow it gets fooled

The final desk call

This is the model you usually want first. It listens to the room instead of trusting one voice.

Default forecast

Ensemble

The table captain

Combines the model stack into one calmer probability.

Helps when

Best when no single model has earned the right to dominate the board.

Gets fooled by

Can still share the same blind spot if every model is missing the same baseball context.

The steady core

These are the workhorse actuarial models. They prefer simple relationships and punish noise.

Core GLM

GLM Ridge

The stabilizer

Keeps every input in the conversation, but quiets the loud ones.

Helps when

Best when a lot of signals are useful but none should get to shout.

Gets fooled by

Can keep too many weak signals alive and make the forecast feel safer than it is.

Core GLM

GLM ENet

The compromise

Blends ridge's patience with lasso's willingness to cut clutter.

Helps when

Best when related baseball signals travel in packs, like pitching, bullpen, and lineup strength.

Gets fooled by

Can mute a real edge if the useful signals are tangled together.

Core GLM

GLM Lasso

The pruner

Deletes weak inputs so the forecast can focus on the strongest evidence.

Helps when

Best when the feature set is crowded and the model needs discipline.

Gets fooled by

Can cut a subtle signal that only matters in certain matchups.

Core GLM

Vanilla GLM

The baseline

The simple no-shrinkage version. It shows what the raw GLM wants to say.

Helps when

Best as a sanity check against the regularized models.

Gets fooled by

Most likely to overreact to small samples or noisy features.

Credibility models

These start with another opinion, then ask how much the data should move away from it.

Opt-in core

Lasso Credibility (Market)

Market-aware credibility

Starts with a clean market opinion and lets lasso decide what the model adds.

Helps when

Best when the betting market is a useful reference, but not the whole answer.

Gets fooled by

Blocked if the market complement is stale, missing, or too close to copying the board.

Opt-in core

Lasso Credibility (Prior)

Prior-aware credibility

Starts with a prior model opinion and learns what today's data should change.

Helps when

Best when there is a trustworthy pregame prior to build from.

Gets fooled by

Can inherit old mistakes if the prior is not audited and frozen correctly.

Theory-compatible challengers

These are still actuarial-adjacent, but they are kept out of the default champion lane until evidence catches up.

Extension

GAM Spline

The curve reader

Lets some relationships bend instead of forcing every input into a straight line.

Helps when

Best when a baseball signal has a sweet spot or threshold.

Gets fooled by

Can draw a beautiful curve through noise and call it signal.

Extension

GLMM Logit

The team-context model

Keeps the main inputs, then gives teams their own background tendencies.

Helps when

Best when team identity matters beyond the measured features.

Gets fooled by

Can confuse team reputation with real current strength.

Extension

DGLM Margin

The margin translator

Thinks about expected run margin first, then translates that into win chance.

Helps when

Best when spread shape says more than winner-only data.

Gets fooled by

Can drift if the run-margin variance is wrong.

Extension

Runs Poisson

The run counter

Treats the game like two run-scoring processes and turns that into a win probability.

Helps when

Best for normal scoring environments where run rates are well behaved.

Gets fooled by

Messy pitching changes, weather, or unusual scoring environments can throw it off.

Experimental scouts

These are useful second opinions. They can spot patterns, but they do not get automatic trust.

Experimental

MARS Hinge Challenger (Proxy)

The bend finder

Looks for sharp turning points where the relationship changes direction.

Helps when

Best for discovering thresholds worth testing later.

Gets fooled by

Can mistake random wiggles for real turning points.

Experimental

Two Stage Challenger

The assembly line

Solves the problem in two steps instead of one big leap.

Helps when

Best when one model should create an intermediate baseball view before the final call.

Gets fooled by

A bad first stage can poison the second stage.

Experimental

Random Forest Challenger

The voting forest

Lets many small decision trees vote on the game.

Helps when

Best for rough interaction hunting.

Gets fooled by

Can look confident because many trees repeated the same noisy split.

Experimental

GBDT Challenger

The booster

Builds a chain of small trees, each trying to fix the last one's misses.

Helps when

Best when there are repeated patterns the GLMs are too simple to catch.

Gets fooled by

Can chase yesterday's mistakes too aggressively.

Experimental

Neural Net Challenger (MLP)

The pattern sponge

A neural model that absorbs many signals and learns flexible combinations.

Helps when

Best as a broad pattern detector when there is enough clean data.

Gets fooled by

Can become a black box that remembers noise instead of baseball truth.

Experimental

Bayes Runs Challenger

The uncertainty counter

A Bayesian run model that tries to carry uncertainty along with the score view.

Helps when

Best when the size of uncertainty matters as much as the point estimate.

Gets fooled by

Can be only as good as the assumptions behind its run process.

Experimental

Bayes BT State Space Challenger

The form tracker

Tracks team strength as something that moves over time.

Helps when

Best when teams are changing and stale season averages lag behind.

Gets fooled by

Can overread short hot streaks or cold streaks.
